# 8btc_scraper
這是一個針對[巴比特論壇]( http://8btc.com/forum-2-1.html)所設計的爬蟲程式。

*其他語言版本: [English](README.md), [繁體中文](README.zh-tw.md)， [简体中文](README.zh-cn.md).*

## 構想
從用戶發帖內容中提取情緒詞語，構建情緒指標，與比特幣期貨走勢擬合，從而構建選擇的策略。

## 爬取的訊息
1. 文章名稱
2. 評論數
3. 閱覽數
3. 發布日期
4. 正面字出現次數 {好，漲，紅，賺}
5. 負面字出現次數 {差，跌，熊，綠，虧}

## 使用工具
Python 2.7, BeautifulSoup4
